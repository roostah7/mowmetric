# MowMetric

Fair, data-backed lawn care pricing — and an open marketplace for verified leads.

MowMetric measures a property via satellite, anchors price to local BLS labor data, and returns an honest benchmark a homeowner can trust. Consumers can optionally opt in to be matched with vetted local providers. Providers buy individual leads from an open, self-serve marketplace — exclusive per lead, verified at submission, priced by quality, decaying by age.

## What's in this repo

- `index.html` — the consumer-facing pricing calculator (static site).
- `mowmetric-business-plan.md` — current business plan and operating model.
